# CaveHill Room Finder

CaveHill Room Finder is a website that provides access for students to determine what rooms are available at what times, on UWI CaveHill Campus.

## Using the application

Visit the website [here](https://cavehill-room-finder.onrender.com).

## Building the application

Building the application locally can be done with:

```sh
bun install
bun start
```
